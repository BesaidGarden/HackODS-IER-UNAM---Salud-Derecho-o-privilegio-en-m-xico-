# Salud en México: ¿Derecho o Lujo?
> **Dashboard interactivo:** [Haz clic aquí para ver el proyecto](https://besaidgarden.github.io/HackODS-IER-UNAM-Salud-Derecho-o-Privilegio-en-Mexico/)

Este proyecto forma parte del **HackODS IER-UNAM** y se centra en el análisis visual de datos para el **ODS 3: Salud y Bienestar**. A través de dashboards interactivos, buscamos evidenciar si la salud en México es un derecho universal o un privilegio condicionado por el acceso a los sistemas de atención y la ubicación geográfica.
El objetivo principal es visibilizar la brecha entre la cobertura teórica y la realidad económica de las familias mexicanas. Queremos demostrar cómo la falta de acceso empuja a los hogares al gasto catastrófico, transformando un derecho básico en un privilegio.

Utilizamos datos oficiales de INEGI, CONEVAL y el SIODS para analizar estas dos métricas:
* **Derecho:** % de la población en México con carencia en servicios de salud (Indicador 3n.1.1).
* **Lujo:** % de hogares con gasto catastrófico en salud (Indicador 3n.1.4).
